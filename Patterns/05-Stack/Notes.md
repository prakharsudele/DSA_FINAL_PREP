# Stack Framework

## Recognition

Use Stack when

- Matching pairs
- Undo operations
- Nested structures
- Most recent unfinished work
- Need Last-In First-Out (LIFO)

---

## Invariant

The stack stores unfinished work.

Whenever work is completed,

remove it from the top.

---

push()

pop()

peek()

isEmpty()
